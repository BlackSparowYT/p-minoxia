Minoxia

What is minoxia?

Minoxia is our project of creating a huge Minecraft amusement park (or in proper English a theme park), we are planning to have multiple zones with different styles, rides, stores and a whole lot more to do. Of course this isn't the first time this has been done BUT this is one of the few times it is done on a server that anyone can join! With this we hope to make an awesome theme park that you guys can enjoy. We will also be looking into bringing a staff mode so you can play as a theme park worker and control rides, look behind the scenes and get exclusive access to stuff! And good news since you all are the first guys to join you will all get access to the closed beta once it gets released :D
